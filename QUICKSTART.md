# Быстрый старт

## Шаги для запуска приложения

1. **Установите зависимости:**
```bash
flutter pub get
```

2. **Настройте URL сервера в `lib/utils/app_config.dart`:**
```dart
static const String baseUrl = 'http://localhost:3000'; // или IP вашего сервера
```

3. **Убедитесь, что backend сервер запущен:**
```bash
cd ../us
npm run dev
```

4. **Запустите Flutter приложение:**
```bash
flutter run
```

## Важные замечания

- Для Android эмулятора используйте `http://10.0.2.2:3000`
- Для iOS симулятора используйте `http://localhost:3000`
- Для физических устройств используйте IP адрес вашего компьютера (например, `http://192.168.1.100:3000`)

## Тестирование

1. Зарегистрируйте нового пользователя
2. Войдите в систему
3. Найдите другого пользователя через поиск
4. Создайте чат и отправьте сообщение
5. Сообщения должны приходить в real-time через WebSocket
